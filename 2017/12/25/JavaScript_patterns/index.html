基本要点

对象的分类？

• 本地对象（Native）：由 ECMAScript 标准规范定义的对象；
• 宿主对象（Host）：由宿主环境创建的对象（比如浏览器环境）。

本地对象：
• 内置对象（比如 Array，Date）；
• 自定义对象（var o = {}）。

宿主对象包含window和所有DOM对象

隐式全局变量与全局变量的区别？

1、可以通过delete运算符删除。

原因：
隐式全局变量并不算是真正的变量，但他们是全局对象的属性成员。属性是可以通过delete运算符删除的，而变量不可以被删除。

（注：在浏览器环境中，所有 JavaScript 代码都是在 window 作用域内的，所以在这种情况下，我们所说的全局变量其实都是 window 下的一个属性，故可以用 delete 删除，但在如 nodejs 或 gjs等非浏览器环境下，显式声明的全局变量无法用 delete 删除。）

javascript引擎的“变量提前”，工作过程是怎样的？

分为两个阶段：
1、创建变量、函数和参数
这一步是预编译的过程，它会扫描整段代码的上下文。
2、代码的运行
这一阶段将创建函数表达式和一些非法的标识符（未声明的变量）。

尽管“变量提前”这个概念并没有在 ECMAScript 标准中定义，但我们常常用它来解释预编译的行为过程。

HTMLCollection对象

HTMLCollection 是由 DOM 方法返回的对象，比如：
• document.getElementsByName()
• document.getElementsByClassName()
• document.getElementsByTagName()

以及
document.images
document.links
document.forms
document.forms[0].elements

HTMLCollection对象是文档中的活动对象。也就是说每次通过它们访问集合的 length 时，总是会去查询 DOM，而 DOM 操作则是很耗资源的。

数字转换？

三者的区别：
1、后两种方法比parseInt()更快一些；
2、“08 hello”这类字符串转换为数字，则必须使用 parseInt()。

原因：
parseInt()是一种“解析”而不是简单的“转换”。
